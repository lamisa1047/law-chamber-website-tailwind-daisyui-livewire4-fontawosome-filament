#  Advocate Portfolio & Consultation Website

A modern legal service platform built with Laravel to help advocates and law firms establish a professional online presence, showcase expertise, publish blogs, and manage client consultations efficiently.

---

##  Project Overview

The Advocate Portfolio & Consultation Website is a full-stack web application consisting of:

- Public Website (Frontend) – For visitors and clients  
- Admin Panel (FilamentPHP) – For managing website content  

It enables advocates to present their professional identity, highlight services, and handle consultation requests in a structured and user-friendly way.

---

##  Features

###  Frontend (Public Website)

####  Home Page
- Hero section with headline
- Professional introduction
- Latest blogs
- Consultation call-to-action

####  About Page
- Biography
- Educational background
- Professional experience
- Achievements

####  Practice Areas
- Legal categories (e.g., Land Law, Criminal Law, Family Law, Corporate Law)
- Individual detail

####  Blog System
- Blog listing page
- Blog details page

####  Consultation Request System
- Whatsapp Intregeted

####  Gallery
- Events
- Title

####  Contact Page
- Contact form
- Address, phone, email
- Google Map integration

---

### ⚙️ Admin Panel (FilamentPHP)

Admin can manage:
- Practice Areas
- Blog Post
- Gallery Images
- About Page Content
- Contact Information
- Website Settings
